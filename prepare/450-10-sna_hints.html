Helpful Hints for SNA in R

You are welcome to use any language process you would like to compute the centrality measures for your homework. But by popular request, here are some helpful hints to get you started with SNA in R.

There are probably better ways to do much of this, and certainly more complete examples on the Internet, but perhaps this can point you in a good direction.

Libraries

I have found these two libraries helpful for plotting and computing social network values in R:

  • igraph
  • sna

Getting from a CSV to a graph

# Reading an edge list mommyEdgesTwitter = read.csv("MommyMentionMap.csv", header = FALSE) twitterGraph = graph.data.frame(mommyEdgesTwitter)

Basic Plotting

# Plot a simple graph using Fruchterman-Reingold layout # Assuming a graph "g" layout1 = layout.fruchterman.reingold(g) layout2 = layout.auto(g) plot(g, layout=layout2)

Be aware that computing layouts and plotting graphs can take a while for large graphs. In my opinion, these basic R libraries are not well-suited for nice layouts of large graphs.

Basic Calculations

# Assuming a graph "g" # Compute density graph.density(g) # Compute centralization measures for all nodes in the graph dc = centralization.degree(g) cc = centralization.closeness(g) bc = centralization.betweenness(g) # For each of the centralization measures, the value for every # node is stored in a list $res (e.g., dc$res)

Remember, the Web is filled with many more examples and tutorials.
