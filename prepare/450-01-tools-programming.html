Good Programming Practice

While it will not be strictly enforced, as in a programming course, you are encouraged to use good style, code conventions, and programming practice.

The Google Python Style Guide is a good guide to base your code on. (Be aware that the textbook author does not adhere to these principles).

Finally, as a matter of good practice, be aware that if you put statements directly in your python files, it causes side effects when your code is imported into other files because that code is run. To avoid this, it is recommended that you structure your files with a main method as follows. This will ensure that your main method is run if your file is invoked directly and not if it is included in another module.

import sys def main(argv): print "Hello World" if __name__ == "__main__": main(sys.argv)
