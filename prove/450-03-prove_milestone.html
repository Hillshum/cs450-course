03 Prove : Milestone - Decision Tree Classifier

Objective

The Decision Tree Assignment can be somewhat challenging. To help you stay on track to completing the assignment, please complete the following portions of the assignment by the day noted in I-Learn.

Instructions

  1. Make sure you are familiar with the instructions for the complete assignment.

  2. Create a class for your decision tree classifier.

  3. Ensure that you can read in at least one dataset with discrete values and send it to your classifier. It is just fine to pre-process the data in another program or script to prepare it, but you should have at least some data for your algorithm.

  4. Have a mechanism for representing your tree, along with branches, leaf nodes, and how it will store attributes and target values.

  5. Have a function that can calculate entropy, and verify that it computes the correct value (check it by hand) for cases with both two classes, and also more than two classes.

  6. Have a recursive function to build a tree that can partition the data it receives according to the value of an attribute. And that can call itself.

  7. Have the logic in place for at least the base cases of the recursive algorithm.

Submission

Answer the questions in the accompanying I-Learn assessment. It asks you to enumerate which of the above are complete.
