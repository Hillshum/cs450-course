01 Prove : Assignment - Experiment Shell & Hardcoded Classifier

Objective

Create an experiment shell that can be used to load data, apply a classification algorithm, and report results. As a placeholder, implement a classifier, "HardCodedClassifier", that always predicts the same class.

More than anything, the purpose of this assignment is to make sure you have the pieces in place to be successful in future assignments. Next week, we will program an actual classifier that will make use of the framework you put in place here.

Instructions

  1. Write a program that can parse data in from a CSV (comma-separated values) file.
  2. Please use the popular Iris dataset, found here: http://archive.ics.uci.edu/ml/datasets/Iris . It contains 150 instances of three classes of Irises (50 per class), and four numeric attributes (in addition to the class name).(While not required for this assignment, be aware that next week you will need to handle nominal attributes as well.)
  3. Split the dataset into two sets: a training set (70%) and a testing set (30%). Don't forget to randomize order of the dataset before splitting.
  4. Create a "HardCoded" classifier that always predicts the first class (e.g., Iris-setosa, but it doesn't really matter which one you choose).
  5. Use your new classifier to try to predict values for the test set, and report its overall accuracy.
  6. Create a new public repository at GitHub, and push your code to it.

You should code this assignment in Python.

Submission

When complete, push your code to a public GitHub repository and answer the questions in the attached PDF. Then submit this PDF to I-Learn.
