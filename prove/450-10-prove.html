10 Prove : Assignment - Social Network Analysis

Objective

Be able to find the central nodes of a social network.

Instructions

Compute the degree centrality, closeness centrality, and betweenness centrality for each node in two different social networks. Then report the node with the highest value of each.

You may use any software you like to identify these values.

The datasets you will be analyzing consist of the network established by Mommy Bloggers linking to one another on their blogs, and mentioning one another in Twitter. In addition, a smaller test dataset is provided for you to start with if that is helpful.

The datasets can be found here:

  • Mommy_blogLinks.csv
  • Mommy_twitterMentions.csv

A small test version for experimenting if you would like is: Mommy_blogLinks_tiny.csv.

These csv files contain a list of directed edges, with the value in the first column representing the source node and the value in the second column representing the destination. For example:

a,b
a,c
b,a

would represent a graph of three nodes: {a, b, c}, with the following edges: (a->b), (a->c), and (b->a).

After computing the centrality values, you are encourage to go above and beyond by further analyzing these social networks to find interesting things, coming up with a meaningful visualization of these networks, or analyzing other additional social networks.

Submission
