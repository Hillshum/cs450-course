06 Prove : Assignment - Neural Network Classifier

Objective

Understand Neural Networks, in particular, back-propagation for multi-layer perceptron networks.

Instructions

Add to your experiment shell from previous assignments. Implement a multi-layer perceptron neural network.

Your assignment this week is to finish the neural network that was started last two weeks, and then to experiment with it on various datasets.

Experiment Guidelines

After implementing the algorithm, use it to classify the following datasets:

  1. Iris
  2. Diabetes (note this dataset is also included with weka)

For each one, experiment with different parameters for the algorithm.

Also, for each one, also use an existing implementation of a multi-layer perceptron network and compare/contrast your results (Note: in Weka this is called MultilayerPerceptron under the "functions" category).

MINIMUM STANDARD REQUIREMENTS

  • Implement the basic MLP algorithm
  • Handle nominal and numeric data (if you choose, nominal data can be handled via pre-processing)
  • Produce a graph of your learner's accuracy on the validation set (or the training set if necessary) per iteration of the algorithm.
  • Basic experimentation
    • Use the supplied datasets
    • Try a few different configurations (# of layers, # of nodes per layer)
    • Produce graphs of the training progress
    • Compare overall results to existing implementations of MLP

OPPORTUNITIES TO GO ABOVE AND BEYOND:

  • Stopping criteria
  • Significant experimentation (Seek out additional datasets, compare and contrast many different configurations, experiment with different strategies for starting weights and normalization, etc.)
  • Regression
  • Different activation functions (e.g., tanh)
  • Implement batch and sequential update, and compare results
  • Using momentum
  • Any other ideas you have

Questions for Consideration

  • How should categorical data be handled?
  • How many layers should be in your network?
  • How many nodes should be in each hidden layer?
  • How many nodes should be in the output layer (e.g., one? one per class?)?
  • When should learning be stopped?
